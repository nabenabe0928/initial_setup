# ctrl+alpha
- enter (`editor.action.insertLineAfter`): add an empty line below
- number (`workbench.action.focus[1-9]EditorGroup`): move to the `n`-th window
- b (`workbench.action.toggleSidebarVisibility`): open the side bar
- j (`workbench.action.togglePanel`): same as ctrl+`
- l (`expandLineSelection`): Select rows
- p (`workbench.action.quickOpen`): open a file
- \ (`workbench.action.splitEditor`): add another window at the right side
- / (`editor.action.commentLine`): Add a comment symbol

# Alt+alpha
- c (`toggleSearchCaseSensitive`): Switch `Match Case` in word search
- z (`editor.action.toggleWordWrap`): Fold lines or not
- up/down (`editor.action.moveLinesUp/DownAction`): Move the current line

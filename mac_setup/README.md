# Key Mapping Setup

Keyboards mostly have three to five keys at the left side of the space key.
Aligning with the MacBook's tiling functionalities, we define the keys from the right side:
- Cmd (Ctrl for Ubuntu),
- Ctrl (Super for Ubuntu),
- Opt (Alt for Ubuntu).

# Disabled Keyboard Shortcuts
- cmd+shift+/: Help menu (Keyboard shortcuts > App Shortcuts)
- cmd+m: Minimize window (Keyboard shortcuts > Windows > General)
